### 🧭 Philosophy

> “정확함보다 **명료함**, 속도보다 **방향**.”

저는 **시스템을 설계하고, 사람을 연결하는 개발자**입니다.  
코드는 단순한 결과물이 아니라, 사고의 흔적이라 생각합니다.  
복잡한 문제를 단순하게 만드는 과정 속에서  
**팀이 성장하고, 더 나은 구조가 만들어지는 순간**을 가장 좋아합니다.
